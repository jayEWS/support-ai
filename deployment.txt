# ==============================================================================
# GUIDE: DEPLOYMENT SUPPORT-AI ON SINGAPORE PC (WINDOWS)
# Location: Singapore | Database: Neon Cloud (PostgreSQL)
# ==============================================================================

### LANGKAH 1: PERSIAPAN FOLDER & KODE
1. Di PC Singapore, buat folder: C:\Project\support-ai (atau lokasi lain).
2. Download/Copy file project dari GitHub atau Flashdisk ke folder tersebut.
3. Pastikan Python 3.10 atau 3.11 sudah terinstall di PC tersebut.

### LANGKAH 2: SETUP VIRTUAL ENVIRONMENT (POWERSHELL)
Buka PowerShell di folder project, lalu jalankan:
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

### LANGKAH 3: CONFIGURASI .ENV (FILE RAHASIA)
Buat file bernama ".env" di folder root project. Isi dengan ini:

DATABASE_URL=postgresql://neondb_owner:npg_4MEQOka9dlTc@ep-floral-art-a1sth6sk-pooler.ap-southeast-1.aws.neon.tech/neondb?sslmode=require
GROQ_API_KEY=gsk_... (Masukkan Key Anda)
GOOGLE_CLIENT_ID=... (Masukkan ID Anda)
GOOGLE_CLIENT_SECRET=... (Masukkan Secret Anda)
SECRET_KEY=support-ai-sg-24h
PORT=8001
ENV=production

### LANGKAH 4: MEMBUAT ONLINE (CLOUDFLARE TUNNEL - GRATIS)
Agar bisa diakses dari luar (Always Online tanpa IP Public):
1. Download 'cloudflared' dari: https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-windows-amd64.msi
2. Install di PC Singapore.
3. Login ke Cloudflare Zero Trust (https://one.dash.cloudflare.com/).
4. Pergi ke "Networks" -> "Tunnels" -> "Create a Tunnel".
5. Pilih nama: "sg-server-support".
6. Jalankan perintah instalasi service yang diberikan Cloudflare di CMD (Administrator).
7. Di Cloudflare Dashboard, arahkan Public Hostname (misal: support.jayews.com) ke:
   Type: HTTP
   URL: localhost:8001

### LANGKAH 5: SCRIPT AUTO-START (ONLINE OTOMATIS)
Gunakan file `NYALAKAN_SERVER.bat` yang sudah saya buat di folder project.
1. Klik kanan `NYALAKAN_SERVER.bat` -> Send to Desktop (Create Shortcut).
2. Setiap kali anda menyalakan PC Singapore, klik file ini di Desktop.
3. Server akan otomatis melakukan update Database (Alembic) dan menjalankan server di Port 8001.

### LANGKAH 6: UPDATE GOOGLE OAUTH REDIRECT URI
1. Pergi ke: https://console.cloud.google.com/apis/credentials
2. Edit OAuth 2.0 Client ID anda.
3. Ubah atau Tambah "Authorized redirect URIs" ke link Cloudflare anda:
   Contoh: https://defend-wednesday-journalist-boot.trycloudflare.com/api/auth/google/callback
   (Ganti link di atas dengan link Permanent/Domain anda di Cloudflare Tunnel).

### LANGKAH 7: SYNC DARI INDO PC KE SG PC
- Selalu Commit & Push dari Indo PC ke GitHub.
- Di SG PC, buka terminal: `git pull origin main` lalu klik file `NYALAKAN_SERVER.bat`.

### LANGKAH 6: UPDATE GOOGLE OAUTH URL
Buka Google Cloud Console, tambahkan URL Cloudflare Anda ke "Authorized redirect URIs":
Contoh: https://support.jayews.com/api/auth/google/callback

### KEAMANAN & TIPS:
- Data aman di Neon Cloud (PostgreSQL), jadi jika PC mati, data tidak hilang.
- Gunakan RAM PC Singapore semaksimal mungkin (Pandas & AI akan jalan kencang).
- Pastikan PC di set "Never Sleep" di Windows Power Settings.
