# POS Troubleshooting Workflows

These workflows represent the most common POS support scenarios.

AI should follow them step-by-step.

------------------------------------------------------------------------

# Workflow: POS Cannot Sync

1 Check Internet Connection 2 Ping Backend API 3 Check Last Sync
Timestamp 4 Verify POS Version 5 Restart POS Service 6 Retry Sync

------------------------------------------------------------------------

# Workflow: Receipt Printer Not Printing

1 Check printer power 2 Verify USB / network connection 3 Check printer
driver 4 Restart printer service 5 Perform test print

------------------------------------------------------------------------

# Workflow: Voucher Cannot Redeem

1 Check voucher code validity 2 Verify expiry date 3 Check usage limit 4
Verify customer eligibility 5 Retry redemption

------------------------------------------------------------------------

# Workflow: Membership Points Missing

1 Verify membership ID attached 2 Check loyalty rules 3 Confirm
transaction completion 4 Recalculate points 5 Sync loyalty system

------------------------------------------------------------------------

# Workflow: POS Application Crash

1 Collect crash logs 2 Check memory usage 3 Restart POS application 4
Clear cache 5 Recommend update
